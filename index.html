📚 Librario Privacy Policy

Informativa sulla Privacy

Ultimo aggiornamento: Marzo 2026  ·  Titolare: Roberto Pino  ·  privacy@robxtech.com

1. Introduzione

Librario è un'app personale per catalogare libri. La presente informativa descrive come vengono gestiti i dati degli utenti in conformità al Regolamento UE 2016/679 (GDPR).

2. Dati raccolti

Librario non raccoglie né trasmette dati personali degli utenti.
Tutti i dati inseriti nell'app vengono salvati esclusivamente sul dispositivo dell'utente e non vengono mai inviati a server esterni.

3. Servizi di terze parti

L'app utilizza i seguenti servizi esterni esclusivamente per la ricerca di informazioni sui libri:

Google Books API

  • Fornitore: Google LLC
  • Utilizzo: ricerca libri per titolo o ISBN, recupero copertine, informazioni bibliografiche e prezzi
  • Dati inviati: titolo del libro o codice ISBN cercato dall'utente
  • Privacy: policies.google.com/privacy

Open Library API

  • Fornitore: Internet Archive
  • Utilizzo: ricerca libri come servizio alternativo a Google Books
  • Dati inviati: titolo del libro o codice ISBN cercato dall'utente
  • Privacy: archive.org/about/terms.php

WhatsApp

  • Utilizzo: invio di messaggi precompilati per solleciti prestiti e condivisione lista desideri tramite protocollo URL nativo
  • Nessuna API — nessun dato trasmesso a WhatsApp dall'app

4. Dati sul dispositivo

I seguenti dati vengono salvati localmente sul dispositivo tramite SQLite e AsyncStorage:

  • Catalogo libri
  • Gestione prestiti
  • Lista desideri
  • Mappa scaffali e posizioni libri
  • Preferenze app (tema, lingua)

Questi dati non lasciano mai il dispositivo, salvo quando l'utente esporta volontariamente un file di backup JSON o CSV tramite la funzione Esporta dell'app.

5. Permessi richiesti

PermessoUtilizzo
📷 FotocameraScanner ISBN e foto copertine libri
🔔 NotificheAvvisi scadenza prestiti

I permessi vengono richiesti solo al momento del primo utilizzo e possono essere revocati nelle Impostazioni del dispositivo.

6. Minori

Librario non raccoglie consapevolmente dati da minori di 13 anni.

7. GDPR — Diritti dell'utente

In conformità al GDPR, l'utente ha diritto a:

  • Accesso — sapere quali dati sono memorizzati
  • Rettifica — correggere i propri dati
  • Cancellazione — eliminare i propri dati
  • Portabilità — esportare i propri dati (funzione Backup JSON disponibile nell'app)

Poiché tutti i dati sono sul dispositivo dell'utente, l'utente ha già il pieno controllo. Per qualsiasi richiesta: privacy@robxtech.com

8. Modifiche alla Privacy Policy

Eventuali modifiche verranno pubblicate su questa pagina con aggiornamento della data in cima al documento.

Privacy Policy

Last updated: March 2026  ·  Developer: Roberto Pino  ·  privacy@robxtech.com

1. Introduction

Librario is a personal app for cataloging books. This policy describes how user data is handled in accordance with EU Regulation 2016/679 (GDPR).

2. Data collected

Librario does not collect or transmit any personal user data.
All data entered in the app is saved exclusively on the user's device and is never sent to external servers.

3. Third-party services

The app uses the following external services solely for book information lookup:

Google Books API

  • Provider: Google LLC
  • Use: book search by title or ISBN, cover images, bibliographic info and prices
  • Data sent: book title or ISBN searched by the user
  • Privacy: policies.google.com/privacy

Open Library API

  • Provider: Internet Archive
  • Use: book search as fallback to Google Books
  • Data sent: book title or ISBN searched by the user
  • Privacy: archive.org/about/terms.php

WhatsApp

  • Use: pre-filled messages for loan reminders and wishlist sharing via native URL protocol
  • No API — no data transmitted to WhatsApp by the app

4. On-device data

The following data is stored locally on the device via SQLite and AsyncStorage:

  • Book catalog
  • Loan management
  • Wishlist
  • Shelf map and book positions
  • App preferences (theme, language)

This data never leaves the device, unless the user voluntarily exports a backup file (JSON or CSV) using the app's Export feature.

5. Permissions requested

Permissions are requested only when the corresponding feature is first used and can be revoked at any time from device Settings.

6. Children

7. GDPR — User rights

In accordance with GDPR, users have the right to:

  • Access — know what data is stored
  • Rectification — correct their data
  • Erasure — delete their data
  • Portability — export their data (JSON Backup feature available in the app)

Since all data is on the user's device, users already have full control. For any request: privacy@robxtech.com
